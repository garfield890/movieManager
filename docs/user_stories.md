User Stories:
1. As a movie lover, I want to save my favorite movies, so I can keep track of them.
2. As a general user, I want to I want my personal movie collection and ratings to be tied to my account, so I can access them anytime I log in anywhere
3. As a big fan of [actors], I want to be able to search by [actors], so I can find all their movie.
4. As a big fan of [directors], I want to be able to search by [directors], so I can find all their movie.
5. As a big fan of good movies, I want to be able to search by rating, so I can find movies people love.
6. As a movie explorer, I want to search movies by genre, so I can quickly find films that match my interests.
7. As a fan of specific release eras, I want to filter movies by release year, so I can discover films from a certain time period.
8. As a busy user, I want to mark movies as watched or unwatched, so I can keep track of what I have already seen.
9. As a general user, I want to be able to organize movies into custom lists, so I know where to find what movie.
10. As a movie connoisseur， I want to rate movies, that is my job.
11. As a person who can't make decision, I want the website be able to random recommend me some movies, so I can make decisions. 
12. As a user with poor memory, I want to able to search movie with just keywords not the entire name, because sometime I forget.
