User Stories:
1. As a movie lover, I want to save my favorite movies, so I can keep track of them.
2. As a general user, I want my personal movie collection and ratings to be tied to my account, so I can access them anytime I log in.
3. As a big fan of [actors], I want to be able to search by [actors], so I can find all their movies I have watched.
4. As a big fan of [directors], I want to be able to search by [directors], so I can find all their movies I have watched.
5. As a big fan of good movies, I want to be able to search by my rating, so I can find which movies I liked.
6. As a movie explorer, I want to search movies I have watched by genre, so I can quickly find my films that matched my interests.
7. As a fan of specific release eras, I want to filter movies I have watched by release year, so I can find my films from a certain time period.
8. As a busy user, I want to mark movies as watched or unwatched, so I can keep track of what I have already seen.
9. As a general user, I want to be able to organize my movies into custom lists, so I know where to find what movie.
10. As a movie connoisseur， I want to rate my movies, that is my job.
11. As a person who can't make decisions, I want the website be able to random recommend me some movies, so I can make decisions. 
12. As a user with poor memory, I want to able to search my movies with just keywords not the entire name, because sometime I forget.
