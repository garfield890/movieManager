Project Pitch: The project will be a backend API that will allow different users to track and manage their favorite movies. Users will be able to create an account, login to their accounts, and search for movies from an external database. Users can then add/update/remove these movies from their watched list which would then be updated on our personal database. Users can also rate the films they have watched on a scale from 0-10 and query films based on different attributes like genre, director, lead actor, personal rating, release year, etc. When asked, the API can also generate personalized stats and create recommendations based on a user's watched list.

Deployed link: https://moviemanager-nh3e.onrender.com
Use /docs to test endpoints.
