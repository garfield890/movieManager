Project Pitch: The project will be a backend API that will allow different users to track and manage their favorite movies. Users will be able to create an account, login to their accounts, and search for movies from our database. Users can then add/update/remove these movies from their watched list which would be updated on their personal account. Users can also rate the films they have watched on a scale from 0-10 and query films based on different attributes like genre, director, and lead actor (personal rating and release year will be added soon). When asked, the API can also generate personalized stats and create recommendations based on a user's watched list.

Our database stores the top 250 movies from IMDb before 2023 (basically any movie with an imdb-rating of 8.1 or higher that was released before April 3rd, 2023).

Deployed link: https://moviemanager-nh3e.onrender.com
Use /docs to test endpoints.
